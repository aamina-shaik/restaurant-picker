# 🍽️ What Should We Eat?

A simple, fun web app that takes the stress out of choosing where to eat. Enter your favourite restaurants, hit one button, and let fate decide!

## 📸 Preview

![App Screenshot](screenshot.png)

## ✨ Features

- Add multiple restaurant names to your list
- One-tap random restaurant selector
- Clean, minimal interface — no sign-up, no fuss
- Works entirely in the browser (no backend needed)
